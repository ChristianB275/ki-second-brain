# 07-Intelligence

Institutionelles Gedaechtnis — Entscheidungen, Meetings, Recherche.

**Unterordner:**
- `decisions/` — Wichtige Entscheidungen mit Kontext und Begruendung (`YYYY-MM-DD-{thema}.md`)
- `meetings/` — Meeting-Notizen, erstellt durch `/os meeting` (`YYYY-MM-DD-{thema}.md`)

**Tipp:** Entscheidungen dokumentieren spart spaeter Stunden. Format: Was entschieden, warum, welche Alternativen verworfen.
