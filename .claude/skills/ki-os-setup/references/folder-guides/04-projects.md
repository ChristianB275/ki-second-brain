# 04-Projects

Jedes aktive Projekt bekommt einen eigenen Unterordner mit `README.md`.

**Struktur pro Projekt:**
```
04-projects/{projektname}/
  README.md       — Ziel, Status, naechste Schritte
  plan-YYYY-MM-DD.md  — Implementierungsplaene (erstellt durch /os plan)
  [weitere Dateien]    — Notizen, Recherche, Drafts
```

**Lebenszyklus:**
- Neues Projekt: `/os plan` oder manuell Ordner anlegen
- Abgeschlossen: Ordner nach `09-reference/` verschieben, Eintrag in `completed-projects.md`

**Tipp:** Das `README.md` ist der Einstiegspunkt. Halte es kurz: Ziel, Status, naechste Schritte.
