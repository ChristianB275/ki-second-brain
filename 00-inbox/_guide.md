# 00-Inbox

Hier landet alles was schnell rein muss. Hauptdatei: `capture.md`.

**Wie nutzen:**
- `/capture` — Claude kategorisiert automatisch (Task, Idee, Loop, Entscheidung)
- Manuell reinschreiben geht auch
- Beim Weekly Review (`/review`) wird die Inbox geleert

**Regel:** Nie direkt sortieren — erst rein, beim Review entscheiden.
